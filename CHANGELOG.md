## [1.0.0] - 2025-12-19

### Added
- Initial release
- Support for MySQL, PostgreSQL, SQLite, SQL Server databases
- Pure PHP database dumping (no external tools required)
- File backup with include/exclude patterns
- AES-256 encryption support
- Multiple storage disk support
- Automatic cleanup with retention policies
- Artisan commands: backup:run, backup:list, backup:clean
- Comprehensive test suite
- Full documentation
